## 파이썬 자료형

### 숫자형

~~~python
# 정수형 : Integer
a = 12
b = 0
c = -34

# 실수형 : Float
a = 1.2
b = -3.4

# 8진수 : Octal
a = 0o20
b = 0o30

# 16진수 : Hexadecimal
a = 0xa1
b = 0x45
~~~

### 연산자

~~~python
# 더하기
> 3 + 5
8

# 빼기
> 9 - 4
5

# 곱하기
> 3 * 3
9

# 제곱
> 3 ** 3
27

# 나누기
> 10 / 4
2.5

# 나누기 나머지
> 9 % 4
1

# 나누기 몫
> 9 // 4
2
~~~
