## collections - 컬렉션
- 컬렉션은 서로 관련된 정보를 그룹화하는데 용이함

### 셋업
- 컬렉션을 사용하려면 먼저 _config.yml에 컬렉션을 정의해야 함
```
# staff_members를 컬렉션으로 정의한 예시
collections:
  - staff_members
```
