## git1

생활코딩 강의의 요약 정리

### git 이란

리눅스를 개발하기 위해 만들어진 분산버전관리시스템

문서의 version(commit)관리를 지원하는 시스템. 버전관리 & 백업 & 협업이 가능

git은 CLI(Git Bash), TortoiseGit, Sourcetree 로 학습&사용 가능

CLI 학습 > TortoiseGit, Sourcetree 사용은 쉬움

TortoiseGit, Sourcetree 학습 > CLI 사용은 어려움

따라서 CLI를 기반으로 학습진행
