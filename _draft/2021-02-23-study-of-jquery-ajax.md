# jquery 이란
- jquery는 javascript library임
- jquery는 다양한 (비표준) 브라우저에서 javascript가 올바르게 작동가능하도록 함
- 비동기식 HTTP Request를 수행하는 함수 ajax가 있음

# jquery ajax 주요 options
- ajax는 옵션들을 object로 만들어 전달
## url
- 기본옵션, Request가 보내져야 하는 URL
